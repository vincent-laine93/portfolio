🏢 Contexte & entreprise

🌐

Gollal — Clichy-sous-Bois (93)

Gollal est une entreprise basée à Clichy-sous-Bois. Dans le cadre de mon stage de première année de BTS SIO, j'ai été chargé de développer from scratch le site web officiel de l'entreprise. Le site est une landing page présentant l'activité de la société, conçue pour être accessible à un public international avec un système de changement de langue intégré.

⚙️ Fonctionnalités développées

🌍 Internationalisation (i18n)

Système de changement de langue via un useState React. Les traductions étaient d'abord stockées dans des fichiers JSON locaux, puis migrées vers Firebase Remote Config pour permettre des mises à jour sans redéploiement.

📱 Responsive Design complet

Menu burger pour mobile, ancrage interne fluide entre les sections, navigation optimisée pour tous les formats d'écran (mobile, tablette, desktop).

♿ Accessibilité

Respect des bonnes pratiques d'accessibilité web : balises sémantiques, contrastes, attributs ARIA et navigation clavier.

🎨 Charte graphique

Intégration fidèle de la charte graphique de l'entreprise : couleurs, typographies, espacements et composants visuels cohérents.

⚡ Firebase Remote Config

Migration des traductions vers Firebase Remote Config pour permettre à l'équipe de modifier le contenu multilingue sans toucher au code source.

🔄 Méthode Agile

Travail en équipe avec méthode Scrum, suivi des tâches sur Jira, versionnement sur GitLab et participation aux réunions de sprint.

👨‍💻 Mon rôle dans le projet

  • Développement complet du site from scratch avec React et TypeScript
  • Mise en place de l'internationalisation avec i18next et fichiers JSON de traduction
  • Migration des traductions vers Firebase Remote Config pour une gestion dynamique du contenu
  • Intégration du responsive design : menu burger, ancrage interne, adaptation mobile
  • Respect de la charte graphique fournie par l'équipe design
  • Application des bonnes pratiques d'accessibilité web
  • Configuration du projet avec Vite et gestion du déploiement sur Firebase Hosting
